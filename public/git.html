git 常用指令总结

版本回退

git reset --hard [commit_ID]

强制提交

git push -f

查看特定文件历史更改记录

git log [path]

查看某次commit的的详情修改

git show [commit_ID]

对某个文件进行撤销git add操作

git reset HEAD XXX.html

提交本地tag

git push origin {tagID}

查看某次commit内容

git show {commit_id}

合并单次commit

git cherry-pick {commit_id}

撤销特定commit 顺序 从新到旧

git revert {最新commitId} {第二commitId} {第三commitId}

推送本地所有tag

git push origin --tags

分支合并

git merge [branch_name]

撤销本地特定文件的修改

git checkout [file_name]

当线上git过大时候 git pull 无法拉去 可以采用下边指令

git pull origin test

删除git空目录

git clean -fd

git 撤销新的未添加到暂存区的文件

git clean -d -fx

查看git命令历史记录

git reflog

撤销git merge(未commit解决冲突前)

git reset --hard HEAD
